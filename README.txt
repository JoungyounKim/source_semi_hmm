
1. main.R

	2-1. A file with functions.

3. HMM-FDR.zip
	3-1. contains functions by Sun&Cai

4. ILI-France.csv: ILI data
	
5. ILI_res.txt
	5-1. record the numerical results

6. ILI_P1.pdf
	6-1. record the estimates of Pr(Z=1)

7. ILI_Z.pdf
